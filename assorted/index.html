Adjustable Standing Desk

This design is a standing desk to be placed on an existing table that is designed for use with a computer. This is intended for people who work all day on their computers, it can be used with a desktop or a laptop. A big issue with standing desks is that they are generally designed for specific heights, this design is adjustable, allowing essentially anyone with any height to use this product. I designed this to be cut from a 3/4″ plywood using a LaserCamm, it can be assembled using only 8 nuts and 3/8″ bolts.
